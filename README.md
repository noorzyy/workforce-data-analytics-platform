# workforce-data-analytics-platform
Authors: Jasnoor Kaur <br>
Currently in progress: <br>

The Workforce Analytics Data Platform is an end-to-end project designed to automate data processing, cleaning, analysis, and visualization workflows.
It simulates a real-world analytics environment by integrating cloud services, data engineering tools, and statistical methods to transform raw workforce or business data into actionable insights.

Tech Stack: Python (Pandas, NumPy, Matplotlib, Seaborn), SQL, Azure, Databricks, Power BI, Excel, PowerShell <br>
Skills to hone: Analysis, data engineering, automation, visualization. <br>
